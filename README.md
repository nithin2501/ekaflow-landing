# EkaFlow Alpha v1.1 Landing Page

High-conversion, vision-to-email engine landing page featuring modern Glassmorphism aesthetics and advanced CSS animations.

## Deployment to GitHub Pages

1. **Create a new repository** on [GitHub](https://github.com/new).
2. **Upload these files**:
   - `index.html`
   - `style.css`
   - `logo.png`
   - `favicon.png`
3. **Enable GitHub Pages**:
   - In your repository, go to **Settings** > **Pages**.
   - Under **Build and deployment** > **Source**, select **Deploy from a branch**.
   - Select the `main` branch and the `/(root)` folder.
   - Click **Save**.

Your site will be live at `https://<your-username>.github.io/<repository-name>/` within a few minutes.
